BelkenAI × TikTok

BelkenAI is an AI content and operations platform. Our TikTok integration lets an authenticated user securely connect their own TikTok account and publish or schedule video content to it directly from the BelkenAI dashboard. We never post without the account owner’s action and authorization.

How it works (end‑to‑end)

  1. Connect: The user clicks “Connect TikTok” in the BelkenAI dashboard and is sent to TikTok’s official Login Kit / OAuth screen, where they review and approve the requested permissions.
  2. Authorize: TikTok returns an access token scoped only to the permissions the user approved. BelkenAI stores this token securely and uses it solely on the user’s behalf.
  3. Create & review: The user uploads or selects a video and writes a caption inside BelkenAI. Nothing is sent to TikTok until the user confirms.
  4. Publish or schedule: On the user’s command, BelkenAI uses the TikTok Content Posting API to upload the video to the user’s own TikTok account — immediately or at a scheduled time the user chooses.
  5. Disconnect anytime: The user can revoke access at any time from the BelkenAI dashboard or from their TikTok account settings, which immediately ends BelkenAI’s access.

Permissions we request — and why

user.info.basic

Read the connected account’s basic profile (display name and avatar) so the user can confirm which TikTok account they are posting to.

video.upload

Upload a video the user created into their own TikTok account as a draft for the user to review and post.

video.publish

Publish or schedule the user’s video to their own TikTok account when the user explicitly chooses to post.

Your data & control

BelkenAI accesses TikTok data only to provide the features above. We do not sell TikTok data, and we do not share it with third parties except the infrastructure providers required to operate the service.
